# 🌟 Stellar Catalogs

## Available Catalogs

### Hipparcos
- **Stars**: 118,000
- **Mission**: ESA Hipparcos (1989-1993)
- **Data**: Positions, parallaxes, proper motions, magnitudes
- **Link**: [hipparcoscatalog.com](https://hipparcoscatalog.com/)

### Gaia DR3
- **Stars**: 1.7 billion
- **Mission**: ESA Gaia (2014-present)
- **Data**: Positions, parallaxes, proper motions, photometry, radial velocities
- **Link**: [cosmos.esa.int/web/gaia](https://www.cosmos.esa.int/web/gaia)

### Bright Star Catalog
- **Stars**: ~9,100 (brightest stars, mag < 6.5)
- **Data**: Positions, magnitudes, spectral types
