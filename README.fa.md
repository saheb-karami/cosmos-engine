## 🔭 موتور کیهان چیست؟

**موتور کیهان** یک آزمایشگاه متن‌باز علم داده برای کاوش در جهان از طریق کدنویسی است. ما داده‌های واقعی نجومی، شبیه‌سازی‌های فیزیکی، و مصورسازی داده را ترکیب می‌کنیم تا کیهان را قابل دسترس، قابل درک و شگفت‌انگیز کنیم.

از شکار سیارات فرازمینی تا فیزیک سیاه‌چاله‌ها، از مکانیک مداری تا مصورسازی تار کیهانی — اینجا جایی است که داده با ستارگان دیدار می‌کند.

| پروژه | توضیح | وضعیت |
|-------|--------|--------|
| 🪐 **شکارچی سیارات فرازمینی** | تحلیل داده‌های Kepler/TESS ناسا | 🟢 فعال |
| 🕳️ **آزمایشگاه سیاه‌چاله** | شبیه‌سازی سیاه‌چاله‌های شوارتزشیلد و کر | 🟡 در حال انجام |

| 🕸️ **تار کیهانی** | مصورسازی ساختار بزرگ‌مقیاس جهان | 📋 برنامه‌ریزی‌شده |

## 🛠️ شروع سریع

```bash
git clone https://github.com/saheb-karami/cosmos-engine.git
cd cosmos-engine
pip install -r requirements.txt
```

### محاسبه سرعت مداری

```python
from tools.orbital_mechanics import OrbitalMechanics

orb = OrbitalMechanics()

# سرعت مداری ایستگاه فضایی بین‌المللی
v_iss = orb.circular_orbit_velocity(altitude_km=408, central_body='Earth')
print(f"سرعت مداری ISS: {v_iss:.2f} km/s")
# خروجی: 7.67 km/s
```
