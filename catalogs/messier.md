# 🌌 Messier Object Catalog

The Messier catalog is a collection of 110 astronomical objects compiled by Charles Messier in the 18th century. Originally intended as a list of "objects to avoid" while hunting comets, it has become one of the most famous catalogs of deep-sky objects.
